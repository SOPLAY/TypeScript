## Yarn Setting

<details>
<summary></summary>
<div markdown="1">

1. 프로젝트 생성

   ```shell
   yarn init -y
   ```

   디렉터리에 `package.json` 이라는 파일 생성

2. 타입스크립트 설정 파일 생성

   typescript 글로벌 설치

   ```
   $ yarn global add typescript
   ```

   이후 프로젝트 디렉터리 안에서 `tsc-init` 명령어를 입력하면 tsconfig.json 파일이 자동생성됨

</details>
